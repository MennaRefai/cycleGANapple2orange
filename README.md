# cycleGANapple2orange
Trained a CycleGAN model for unpaired image-to-image translation between two visual domains (apples ↔ oranges). Implemented the training pipeline . including generator and discriminator architectures, cycle consistency loss, and adversarial training. Monitored training progress and evaluated generated image quality for domain conversion.
